# 测试图片清单

## 一、目录与复现方式

本机测试集位于 Git 忽略目录 `test/final/`：

```text
test/final/
├── synthetic/    # 精确 RGB、分辨率和期望结果的合成夹具
├── real/         # 从本地 test_pic 复制的五张真实图，不上传
└── public/       # 标明来源与许可的公开格式样例
```

在 macOS 上重新生成：

```bash
./scripts/prepare_final_test_images.sh
```

安装全部最终测试图片到当前唯一连接的 Android 设备：

```bash
./scripts/install_test_images.sh test/final
```

如连接多台设备，先设置 `ANDROID_SERIAL=<设备编号>`。图片和截图不提交 Git；生成脚本、哈希和期望结果提交，保证其他组员可以复现。
